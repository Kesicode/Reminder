# User Stories

## Epic 1: Authentication & Profiles
- As a new user, I want to sign up using my email and password so that I can create an account.
- As a returning user, I want to log in so that I can access my reminders.
- As a user who forgot their password, I want to request a password reset email so that I can regain access to my account.
- As a user, I want to edit my profile details (name, avatar) so that my group members can easily identify me.

## Epic 2: Personal Reminders
- As a user, I want to quickly add a new reminder from the dashboard so that I don't forget it.
- As a user, I want to set a due date, time, and priority level for a reminder so that I can organize my tasks.
- As a user, I want to set a recurrence rule (e.g., daily) for my morning routine so that it generates automatically.
- As a user, I want to view my reminders grouped by "Today", "Upcoming", and "Overdue" to prioritize my day.
- As a user, I want to mark a reminder as completed so that it is moved to the completed view.

## Epic 3: Group Management
- As a user, I want to create a new group and invite colleagues via email so that we can collaborate.
- As a Group Owner, I want to promote a member to Group Admin so that they can help manage the group.
- As a Group Admin, I want to remove inactive members from the group.
- As a user, I want to view a list of all groups I belong to and easily switch between them.

## Epic 4: Shared Reminders
- As a group member, I want to create a reminder and assign it to another specific group member.
- As a group member, I want to restrict visibility of a specific shared reminder to only the assigned member and admins, so that other members cannot see it.
- As a group member, I want to see an activity feed of who created or completed reminders recently in the group.
- As a group member, I want to see real-time updates when someone else completes a shared reminder so we avoid duplicate work.

## Epic 5: Notifications
- As a user, I want to receive a push notification when a reminder is assigned to me so I can act on it immediately.
- As a user, I want to see a red badge on the notification bell when I have unread notifications.
- As a user, I want to view all past notifications in a dedicated drawer and mark them as read.

## Epic 6: Calendar
- As a user, I want to see a monthly calendar view showing dots for days that have reminders.
- As a user, I want calendar dots to be color-coded by priority (Green, Amber, Red).
- As a user, I want to click on a calendar date to instantly view all reminders for that day.

## Epic 7: Administration (Super Admin)
- As a Super Admin, I want to view global platform statistics (Total Users, Groups, Reminders) to monitor growth.
- As a Super Admin, I want to search for a specific user and disable their account if they violate terms.
- As a Super Admin, I want to review all groups and delete inactive ones to free up database resources.
